## 🌟 რატომ ეს აპლიკაცია?

ეს რუკა არ არის მხოლოდ კოორდინატების ერთობლიობა, ის შექმნილია უბნის მაცხოვრებლებისთვის და მათთვის, ვინც აქ ცხოვრებას გეგმავს:

*   🏠 **მაცხოვრებლებისთვის:** იპოვეთ უახლოესი მაღაზია, აფთიაქი, ავტოსერვისი ან სილამაზის სალონი.
*   🎓 **განათლება:** ინფორმაცია მასწავლებლებზე (ენები, მუსიკა), ბაღებსა და სკოლებზე.
*   🛒 **პირადი საქმიანობა:** პლატფორმა მათთვის, ვისაც სურს საკუთარი შემოქმედების ან მომსახურების უბნის შიგნით გაზიარება.
*   📈 **დინამიურობა:** მონაცემები ახლდება ბევრად უფრო სწრაფად, ვიდრე სხვა გლობალურ პლატფორმებზე.

## 🚀 ფუნქციები (Features)

*   🗺️ **OSM ინტეგრაცია:** დინამიური რუკა OpenStreetMap-ის გამოყენებით.
*   💾 **Multi-Platform Data:** Geopackage (Mobile/Windows) და GeoJSON (Web) მხარდაჭერა.
*   🛰️ **GPS ნავიგაცია:** მომხმარებლის რეალურ დროში ლოკალიზაცია.
*   🔍 **ჭკვიანი ფილტრაცია:** POI შრეების მართვა და სწრაფი ძებნა.
*   🔗 **Smart Links:** პირდაპირი გადასვლა Google Maps-ზე შეფასებების სანახავად.
